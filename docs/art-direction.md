# 《群英乱斗棋》AI 美术规格 v1

## 统一风格

- 风格：热血华丽的国潮卡牌。
- 构图：半身人物卡牌，角色占画面 70%，面向镜头或 3/4 侧身。
- 画面比例：头像资源建议 1:1；角色宣传图可用 3:4。
- 质感：精细插画、清晰五官、华丽服饰、强轮廓光、金红青绿点缀。
- 背景：抽象朝代纹样、云纹、山海纹、战旗、星火，不要真实复杂场景。
- 禁止：现代服装、写实影视脸、已有游戏 IP 风格、文字、水印、Logo。
