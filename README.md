# 群英乱斗棋

中国历史和神话题材的单人 roguelike 自走棋原型。当前先做规则核心，后续再接微信小游戏或 Cocos Creator 表现层。

## 当前进度

- 已建立纯 TypeScript 规则核心，界面层后续只负责展示和交互。
- 已配置 24 个英雄、10 件装备、6 个圣物，以及 AI 美术生成规格。
- 已实现三种伤害类型：物理、魔法、神话。
- 神话伤害不吃物理和法术减伤，但可以暴击。
- 已实现 30 秒自动战斗模拟，战斗中不能操作，准备阶段才能调整阵容。
- 英雄有 HP，随机环境伤害可以让脆弱英雄本关死亡，下关自动恢复。
- 已接入核心羁绊效果：
  - 异士：降低非异士羁绊触发人数。
  - 武将：提高普攻和物理技能伤害。
  - 盛唐、刺客：提高暴击和暴击伤害。
  - 秦汉：提高物理伤害。
  - 大宋：降低环境伤害。
  - 神话：开局获得初始法力。
  - 山海：根据本关击杀叠加伤害。
  - 文臣：提高回蓝效率。
  - 帝王：提高最终伤害。
- 已实现普通关胜负金币、利息、野怪逐只掉钱、Boss 奖励节点、Boss 额外掉装和装备重铸等级。
- 已实现 5 格商店、1 到 8 人口概率、有限棋子池、买入、出售、备战席、上阵下阵、三合一升星。
- 已实现每关自然经验、4 金买 4 经验、1 到 8 人口升级曲线。
- 已实现 `GameSession` 单局流程：开局、商店购买、上阵、战斗、金币经验结算、进入下一关。
- 已实现装备系统：装备宝箱三选一、库存、穿戴、卸下、每英雄最多 3 件。
- 已实现 Boss 后装备重铸：投入装备后按等级生成新宝箱，每次机会只能使用一次。
- 山海残卷已接入战斗，可以把穿戴者技能伤害类型改为神话。
- 已实现圣物宝箱三选一、永久生效、重复圣物不重复添加。
- 已接入圣物效果：聚宝盆额外金币、招贤榜首次刷新折扣、太学令额外经验、续命符失败减伤。

## 命令


## 目录

- `src/rules.ts`：核心规则和战斗模拟。
- `src/economy.ts`：金币、关卡奖励、Boss 节点和重铸规则。
- `src/shop.ts`：商店、有限棋子池、买卖、升星、经验和人口。
- `src/session.ts`：把规则、经济、商店串成一局游戏流程。
- `src/equipment.ts`：装备宝箱、重铸和装备工具函数。
- `src/relic.ts`：圣物宝箱和圣物工具函数。
- `src/content.ts`：英雄、装备、圣物配置。
- `src/*.test.ts`：规则和配置测试。
- `docs/art-direction.md`：GPT 图像生成用的美术规格。
